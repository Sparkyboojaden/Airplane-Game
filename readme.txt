============readme==================

1. Type "make"
2. Run the file ex1	
3. When the file begins running, you will be prompted with 5 options. 
4. Enter the value (1-5) of whichever task you wish to run. 
-"1" creates a new bomber and begins counting down the fuel
-"2" will then prompt you for a N value to bomb with. 
-"3" prints out a list of bombers N values
-"4" will then prompt you for a N value to refuel.
-"5" will quit the program
